How Illustrator arranges classes

for each object you create, illsutrator assigns a class to it: .st0, .st1, .st2 etc.

objects with same color share the same class, so it's the best way to group it, rounding to whoever comes first (so if you assign .st5 with same color as .st17 for instance, both become .st5).

Class properties

fill, fill-opacity, stroke, stroke-width, stroke-opacity, stroke-dasharray
